Shape Parameters

Modify the shape parameters manually. Press Compute Shape to render.

How to use
  1. Shape can be edited using the toolbar action buttons.
  2. Right-click inside a circle to edit its parameters.
  3. Right-click outside all circles to edit all at once.
  4. Drag inside a circle to reposition it.
  5. Drag outside circles to move the entire shape.
  6. Mouse wheel to zoom.

Button Reference

  • Compute Shape — Recalculates and renders the shape.
  • Export Shape — Save shape to a .txt file.
  • Upload Shape — Load a shape from a .txt file.
  • Preview — Clean view without guides.
  • Mid-segment circle — Click a segment to insert a circle.
  • Add circle — Click anywhere to place a circle.
  • Erase circle — Click a circle to remove it.
  • Add segment — Click two circles to connect them.
  • Remove segment — Click a segment to delete it.
  • Random shape — Generate a random SVG figure.
  • Similar shape — Generate a variation of the current shape.
About

C++ shape generation compiled to WebAssembly via Emscripten. Interactive editing powered by D3.js.
